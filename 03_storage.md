イメージ及び Glance の使い方
----------------------------

### イメージサービス Glance の操作と機能

* glance-api

    ユーザからのリクエストを受けつ受ける REST API のサーバー機能に加えて、実際にテンプレーtのイメージやスナップショットの保存、取り出し処理までを行う。

* glance-registry
    
    格納されたテンプレートイメージやスナップショットの情報をデータベースに記録する。
